# Sistema de Reforço Escolar

Sistema para gerenciamento de alunos, cursos, matrículas e atendimentos.

## Tecnologias

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MySQL

## Funcionalidades

- Login
- Cadastro de alunos
- Cadastro de cursos
- Matrículas
- Agendamento de atendimentos
- Dashboard
- Perfil do usuário
